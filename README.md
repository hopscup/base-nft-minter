# base-nft-minter
Installation: npm install.
Compilation: npx hardhat compile.
Tests: npx hardhat test.
Deployment to Testnet: npx hardhat run scripts/deploy.js --network base-sepolia.
Verification: npx hardhat verify --network base-sepolia <contract-address>.
Minting: Add a script or use ethers.js to call mintNFT.
